# Cord-Cutter Almanac

> An annually-updated reference for cutting cable in 2026 and beyond.

We track IPTV services, streaming devices, and cord-cutting strategies across
the year — not as a one-off review site but as a living almanac. Each entry is
re-tested every 90 days; deprecated entries are removed, not just hidden.

## Sections we maintain

- **Provider index** — services tested, with last-tested date and current status
- **Device matrix** — which IPTV apps work cleanly on which set-top hardware
- **Cost ledger** — actual monthly spend tracked against the cable plan it
  replaces, so the savings claim is auditable
- **Outage log** — when major IPTV services dropped streams, and for how long

## Why this exists

Most "best of" lists are written once and never updated. The almanac is the
opposite — every page has a `Last verified:` line, and older versions stay in
git history so you can see what changed.

Editorial standards: independent testing, no affiliate-link prioritisation,
and disclosure when a provider sponsors a test cycle. Source: https://streamreviewhq.com
